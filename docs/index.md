# Välkommen

Det här är kursboken för uppsatskursen. Den fungerar som ryggrad genom hela
kursen: här hittar du tidsplan, krav, instruktioner, mallar och resurser
samlade på ett ställe.

Använd menyn för att navigera mellan avsnitten.

!!! note "Aktuell kursomgång"
    Innehållet gäller **{{ aktuell_termin }}** (läsår {{ lasar }}).
    Boken uppdateras löpande och byggs automatiskt från källtexten på GitHub.
